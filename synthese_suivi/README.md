# Documentation du module
## Objectifs
Ce module a pour objectif de centraliser les résultats de dénombrements annuels d'oiseaux, à l'échelle départementale.
L'échelle de saisie et de restitution est la commune mais il est possible de joindre des tableaux avec des dénombrements plus précis (commune, secteurs...) mais actuellement ils doivent être en format PDF.
Les espèces ou groupes d'espèces visés sont (liste non exhaustives) :
- Ardéidés
- Cormorans
- Laridés
- Rapaces
- Hirondelles
- Bécassines
- Marouettes
- Cigognes
- Guifettes
- Outarde
- Courlis
- Oedicnème
- Grue
- ...
